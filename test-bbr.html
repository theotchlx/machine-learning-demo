# Test de normalit (Shapiro-Wilk)
shapiro.test(winequality_white$quality)
## 
##  Shapiro-Wilk normality test
## 
## data:  winequality_white$quality
## W = 0.88904, p-value < 2.2e-16
# Crer une variable binaire pour la qualit
winequality_white$quality_binary <- ifelse(winequality_white$quality >= 6, "Good", "Bad")

# Effectuer le test t de Student
t.test(`alcohol` ~ quality_binary, data = winequality_white)
## 
##  Welch Two Sample t-test
## 
## data:  alcohol by quality_binary
## t = -32.495, df = 4389.8, p-value < 2.2e-16
## alternative hypothesis: true difference in means between group Bad and group Good is not equal to 0
## 95 percent confidence interval:
##  -1.0596430 -0.9390554
## sample estimates:
##  mean in group Bad mean in group Good 
##            9.84953           10.84888
# Sparer les donnes en ensembles d'entranement et de test
set.seed(123)
trainIndex <- createDataPartition(winequality_white$quality, p = .8,
                                  list = FALSE,
                                  times = 1)
wineTrain <- winequality_white[ trainIndex,]
wineTest  <- winequality_white[-trainIndex,]

dim(wineTrain)
## [1] 3919   13
dim(wineTest)
## [1] 979  13
# Entraner le modle de forts alatoires
model <- randomForest(quality ~ ., data = wineTrain)

# Prdire sur l'ensemble de test
predictions <- predict(model, wineTest)

# valuer le modle
eval <- postResample(pred = predictions, obs = wineTest$quality)

Description

RMSE

Le RMSE mesure lcart moyen entre les valeurs prdites par le modle et les valeurs relles. Une valeur de RMSE plus faible indique un meilleur ajustement du modle aux donnes. Ici, la valeur est de 0.4221, ce qui signifie quen moyenne, les prdictions du modle scartent de 0.4221 units des valeurs relles.

Rsquared

Le coefficient de dtermination, ou R-squared, reprsente la proportion de la variance de la variable dpendante qui est prvisible partir des variables indpendantes. Un R-squared de 1 signifie que le modle explique parfaitement la variance des donnes, tandis quun R-squared de 0 signifie que le modle nexplique aucune variance. Ici, la valeur est de 0.7763, indiquant que le modle explique environ 77.63% de la variance dans la qualit du vin.

MAE

Le MAE mesure lerreur moyenne absolue entre les prdictions et les valeurs relles. Contrairement au RMSE, il ne pnalise pas autant les grandes erreurs. Ici, la valeur est de 0.2822, ce qui signifie quen moyenne, les prdictions du modle scartent de 0.2822 units des valeurs relles.
