## 
##  Shapiro-Wilk normality test
## 
## data:  winequality_white$quality
## W = 0.88904, p-value < 2.2e-16
# Créer une variable binaire pour la qualité
winequality_white$quality_binary <- ifelse(winequality_white$quality >= 6, "Good", "Bad")

# Effectuer le test t de Student
t.test(`alcohol` ~ quality_binary, data = winequality_white)
## 
##  Welch Two Sample t-test
## 
## data:  alcohol by quality_binary
## t = -32.495, df = 4389.8, p-value < 2.2e-16
## alternative hypothesis: true difference in means between group Bad and group Good is not equal to 0
## 95 percent confidence interval:
##  -1.0596430 -0.9390554
## sample estimates:
##  mean in group Bad mean in group Good 
##            9.84953           10.84888
# Séparer les données en ensembles d'entraînement et de test
set.seed(123)
trainIndex <- createDataPartition(winequality_white$quality, p = .8,
                                  list = FALSE,
                                  times = 1)
wineTrain <- winequality_white[ trainIndex,]
wineTest  <- winequality_white[-trainIndex,]

dim(wineTrain)
## [1] 3919   13
dim(wineTest)
## [1] 979  13
# Entraîner le modèle de forêts aléatoires
model <- randomForest(quality ~ ., data = wineTrain)

# Prédire sur l'ensemble de test
predictions <- predict(model, wineTest)

# Évaluer le modèle
eval <- postResample(pred = predictions, obs = wineTest$quality)

Description

RMSE

Le RMSE mesure l’écart moyen entre les valeurs prédites par le modèle et les valeurs réelles. Une valeur de RMSE plus faible indique un meilleur ajustement du modèle aux données. Ici, la valeur est de 0.4221, ce qui signifie qu’en moyenne, les prédictions du modèle s’écartent de 0.4221 unités des valeurs réelles.

Rsquared

Le coefficient de détermination, ou R-squared, représente la proportion de la variance de la variable dépendante qui est prévisible à partir des variables indépendantes. Un R-squared de 1 signifie que le modèle explique parfaitement la variance des données, tandis qu’un R-squared de 0 signifie que le modèle n’explique aucune variance. Ici, la valeur est de 0.7763, indiquant que le modèle explique environ 77.63% de la variance dans la qualité du vin.

MAE

Le MAE mesure l’erreur moyenne absolue entre les prédictions et les valeurs réelles. Contrairement au RMSE, il ne pénalise pas autant les grandes erreurs. Ici, la valeur est de 0.2822, ce qui signifie qu’en moyenne, les prédictions du modèle s’écartent de 0.2822 unités des valeurs réelles.
